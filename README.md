# cub3D
## 42 project to make 3D with ray casting
## attention
`It only works on Linux.`
## ___usage___
```c:title
make
```
### ___run___
```c:title
./cub3D cubfiles/*.cub
```
### ___save BMP___
```c:title
./cub3D cubfiles/*.cub --save
```
## cubfiles can be edited.<br>

| identifier | information |
| :-- | :-- |
| R | Resolution |
| NO | North texture |
| SO | South texture |
| WE | West Texture |
| EA | East Texture |
| S | Sprite Texture |
| F | Floor Color |
| C | Ceiling Colors |
|MAP|0 empty space<br> 1 wall<br>2 item<br> N,S,E,W players start potision|
